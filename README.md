# waifubot
Telegram Waifu Bot Using Nodejs

How to install:

Using Termux=
-apt-get update,
-apt-get upgrade,
-apt-get install nodejs,
-apt-get install git,
-git clone https://github.com/ZxOneID/waifubot,
-cd waifubot,
-npm install,
-npm start

Using windows=
-cd waifubot,
-npm install,
-npm start,
